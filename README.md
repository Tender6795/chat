

## Getting Started
1. You should have a postgresql database named 'chat'
2. Get env files from the developer and insert them into the project

3. Run the server:

```bash
cd server/
npm i
npm run start:dev
```
4. Run client:
```bash
cd client/
npm i
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
